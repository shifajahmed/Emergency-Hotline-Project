# Emergency-Hotline-Project 🚨

🔗 **Live Demo:** https://shifajahmed.github.io/Emergency-Hotline-Project/

## Overview

**Emergency-Hotline-Project** is a responsive web application that provides instant access to essential emergency contacts. Designed for efficiency and simplicity, it helps users connect with local emergency services—police, ambulance, fire department—without delay.

> ⚠️ Quick, easy, and reliable access during critical situations.

## Features

- 📞 Quick access to local emergency numbers
- 🗂 Categorized contacts for easy navigation
- 🔍 Search functionality for instant lookup
- 📱 Fully responsive design for mobile and desktop
- 🚀 User-friendly interface for immediate action

## Technologies Used

- **Frontend:** HTML, CSS, JavaScript
- **Libraries/Frameworks:** Tailwind CSS
- **Deployment:** GitHub Pages
